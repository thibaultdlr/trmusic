Embed the Bandcamp Player into your website pages using a widget or the post editor shortcode.

== Description ==

With this plugin you can easily embed a Bandcamp Player into your website using 
a widget, an editor shortcode or both.

You can choose which type of layout you want to use and the colors for 
background and links for each single player.

You can also embed more players just typing different album/track IDs using 
commas to separate them.


== Installation ==

1. Unpack the plugin into `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
